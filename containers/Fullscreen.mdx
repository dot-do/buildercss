# Fullscreen

A fullscreen container for immersive experiences.

## Semantic Structure

```html
<article aria-label="Fullscreen content" data-fullscreen>
  <header>
    <button aria-label="Exit fullscreen">Ã—</button>
  </header>
  <main>
    <!-- Content -->
  </main>
  <footer>
    <!-- Controls -->
  </footer>
</article>
```

## CSS

```css
[data-fullscreen] {
  position: fixed;
  inset: 0;
  z-index: 100;
  background: var(--background);
  display: flex;
  flex-direction: column;
}

[data-fullscreen] > header {
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  z-index: 10;
  padding: var(--spacing-4);
  background: linear-gradient(to bottom, rgba(0,0,0,0.5), transparent);
  opacity: 0;
  transition: opacity 0.3s;
}

[data-fullscreen]:hover > header,
[data-fullscreen]:focus-within > header {
  opacity: 1;
}

[data-fullscreen] > main {
  flex: 1;
  display: flex;
  align-items: center;
  justify-content: center;
  overflow: hidden;
}

[data-fullscreen] > footer {
  position: absolute;
  bottom: 0;
  left: 0;
  right: 0;
  z-index: 10;
  padding: var(--spacing-4);
  background: linear-gradient(to top, rgba(0,0,0,0.5), transparent);
  opacity: 0;
  transition: opacity 0.3s;
}

[data-fullscreen]:hover > footer,
[data-fullscreen]:focus-within > footer {
  opacity: 1;
}

/* Dark theme override for media */
[data-fullscreen][data-theme="dark"] {
  background: black;
  color: white;
}
```

## Use Cases

### Image Gallery
```css
[data-fullscreen] > main img {
  max-width: 100%;
  max-height: 100%;
  object-fit: contain;
}
```

### Video Player
```css
[data-fullscreen] > main video {
  width: 100%;
  height: 100%;
  object-fit: contain;
}
```

### Presentation
```css
[data-fullscreen] > main {
  overflow-x: auto;
  scroll-snap-type: x mandatory;
}
```

## Mobile (React Native)

```tsx
<Modal visible={visible} animationType="fade" statusBarTranslucent>
  <SafeAreaView style={styles.container}>
    <TouchableOpacity style={styles.closeButton} onPress={onClose}>
      <Icon name="close" />
    </TouchableOpacity>
    <View style={styles.content}>{children}</View>
    {controls && <View style={styles.controls}>{controls}</View>}
  </SafeAreaView>
</Modal>
```

## Props

```typescript
interface FullscreenProps {
  open: boolean
  onClose: () => void
  showControls?: boolean
  autoHideControls?: boolean
  theme?: 'light' | 'dark'
  children: ReactNode
  controls?: ReactNode
}
```
