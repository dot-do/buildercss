# DashboardLayout

A full application shell layout for dashboards and admin interfaces.

## Semantic Structure

```html
<body>
  <header>
    <nav>
      <Logo />
      <Navigation />
      <UserMenu />
    </nav>
  </header>

  <main>
    <aside>
      <nav aria-label="App navigation">
        <Sidebar />
      </nav>
    </aside>

    <section>
      <header>
        <PageTitle />
        <Actions />
      </header>
      <article>
        <Slot />
      </article>
    </section>
  </main>
</body>
```


## Slots

| Slot | Element | Required |
|------|---------|----------|
| Logo | `header nav > :first-child` | Yes |
| Navigation | `header nav > ul` | No |
| UserMenu | `header nav > :last-child` | No |
| Sidebar | `aside nav` | Yes |
| PageTitle | `section > header h1` | Yes |
| Actions | `section > header menu` | No |
| Slot | `section > article` | Yes |
