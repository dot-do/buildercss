# StackedLayout

A simple vertical layout for content-focused pages. Header and footer are optional.

## Semantic Structure

```html
<body>
  <header>
    <nav>
      <Logo />
      <Navigation />
    </nav>
  </header>

  <main>
    <Slot />
  </main>

  <footer>
    <FooterContent />
  </footer>
</body>
```

## CSS

```css
body {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

body > header {
  position: sticky;
  top: 0;
  z-index: 50;
  border-bottom: 1px solid var(--border);
}

body > main {
  flex: 1;
  width: 100%;
  max-width: var(--container-width, 1200px);
  margin: 0 auto;
  padding: var(--spacing-6) var(--spacing-4);
}

body > footer {
  border-top: 1px solid var(--border);
  padding: var(--spacing-6) var(--spacing-4);
}
```

## Slots

| Slot | Element | Required |
|------|---------|----------|
| Logo | `header nav > :first-child` | Yes |
| Navigation | `header nav > ul` | No |
| Slot | `main` | Yes |
| FooterContent | `footer` | No |

## Mobile (React Native)

```tsx
<SafeAreaView style={{ flex: 1 }}>
  <Header />
  <ScrollView style={{ flex: 1 }}>
    {children}
  </ScrollView>
  <Footer />
</SafeAreaView>
```
