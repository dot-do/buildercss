# Chat

A chat view for messaging interfaces. Can be used fullscreen, in a slideover, or as a popup widget.

## Semantic Structure

```html
<article aria-label="Chat">
  <header>
    <figure><img src="avatar.jpg" alt="" /></figure>
    <hgroup>
      <h1>AI Assistant</h1>
      <p>Online</p>
    </hgroup>
    <menu>
      <button aria-label="Settings">⚙</button>
      <button aria-label="Close">✕</button>
    </menu>
  </header>

  <section aria-label="Messages">
    <article data-role="user">
      <p>Hello, how can I help?</p>
      <time>10:30 AM</time>
    </article>
    <article data-role="assistant">
      <p>I'm here to assist you with any questions.</p>
      <time>10:31 AM</time>
    </article>
  </section>

  <footer>
    <form>
      <textarea placeholder="Type a message..." rows="1"></textarea>
      <button type="submit" aria-label="Send">→</button>
    </form>
  </footer>
</article>
```
