# Editor

A code editor view using Monaco editor for code/MDX editing.

## Semantic Structure

```html
<article aria-label="Editor">
  <header>
    <nav aria-label="File">
      <span>src/</span>
      <span>components/</span>
      <strong>Button.tsx</strong>
    </nav>
    <menu>
      <button>Format</button>
      <button>Save</button>
      <button>Run</button>
    </menu>
  </header>

  <main>
    <aside aria-label="File explorer">
      <nav><!-- File tree --></nav>
    </aside>

    <section aria-label="Editor">
      <div id="monaco-editor"></div>
    </section>

    <aside aria-label="Output">
      <nav>
        <button aria-pressed="true">Console</button>
        <button>Problems</button>
        <button>Terminal</button>
      </nav>
      <pre><!-- Output --></pre>
    </aside>
  </main>

  <footer>
    <span>TypeScript</span>
    <span>UTF-8</span>
    <span>Ln 42, Col 15</span>
  </footer>
</article>
```

## Monaco Integration

```tsx
import * as monaco from 'monaco-editor'

monaco.editor.create(document.getElementById('monaco-editor'), {
  value: content,
  language: 'typescript',
  theme: 'vs-dark',
  automaticLayout: true,
})
```
