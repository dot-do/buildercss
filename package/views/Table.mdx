# Table

A data table view with sortable columns, filtering, and pagination.

## Semantic Structure

```html
<section aria-label="Data table">
  <header>
    <h1>Users</h1>
    <menu>
      <input type="search" placeholder="Search..." />
      <button>Export</button>
      <button>Add User</button>
    </menu>
  </header>

  <table>
    <thead>
      <tr>
        <th><input type="checkbox" aria-label="Select all" /></th>
        <th><button>Name <span>↕</span></button></th>
        <th><button>Email <span>↕</span></button></th>
        <th><button>Role <span>↕</span></button></th>
        <th><button>Status <span>↕</span></button></th>
        <th>Actions</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td><input type="checkbox" /></td>
        <td>John Doe</td>
        <td>john@example.com</td>
        <td>Admin</td>
        <td><span data-status="active">Active</span></td>
        <td>
          <menu>
            <button>Edit</button>
            <button>Delete</button>
          </menu>
        </td>
      </tr>
    </tbody>
  </table>

  <footer>
    <p>Showing 1-10 of 100</p>
    <nav>
      <button>Previous</button>
      <button>Next</button>
    </nav>
  </footer>
</section>
```
