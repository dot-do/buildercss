    Getting Started with Waitlist.do

    Learn how to set up your first waitlist in minutes.

    Tutorial Getting Started

    Welcome to Waitlist.do! This guide will help you set up your first waitlist in minutes.

    Why Use a Waitlist?

    Waitlists are a powerful tool for building anticipation and gathering early interest in your product or service. They help you:

    • Validate demand before investing heavily in development
    • Build an engaged audience of early adopters
    • Create urgency and exclusivity around your launch
    • Collect valuable data about your potential customers

    Setting Up Your First Waitlist

    Follow these simple steps to get started:

    1. Clone the repository
    2. Install dependencies with pnpm install
    3. Start the development server with pnpm dev
    4. Customize your waitlist configuration

    Customizing Your Configuration

    Edit your configuration file to match your brand:

    export const config = {
  metadata: {
    title: "Join the Waitlist",
    description: "Be first to access our platform"
  },
  hero: {
    title: "Build something amazing",
    subtitle: "Get early access today"
  }
}

    Next Steps

    Once you've set up your basic waitlist, you can customize it with different layouts, add highlights, and integrate with your email service provider.

    The best waitlist pages are simple, clear, and focused on capturing email addresses with minimal friction.
