# Hero

A hero section for landing pages with headline, description, and call-to-action.

## Semantic Structure

```html
<section aria-label="Hero">
  <small>Tagline</small>
  <h1>Main Headline</h1>
  <p>Supporting description text that explains the value proposition.</p>
  <form>
    <input type="email" placeholder="Enter your email" />
    <button type="submit">Get Started</button>
  </form>
  <aside>
    <small>Join 10,000+ users</small>
  </aside>
</section>
```
