# Logos

A logo cloud section for displaying customer or partner logos.

## Semantic Structure

```html
<section aria-label="Trusted by">
  <p>Trusted by leading companies</p>
  <ul>
    <li><img src="logo1.svg" alt="Company 1" /></li>
    <li><img src="logo2.svg" alt="Company 2" /></li>
    <li><img src="logo3.svg" alt="Company 3" /></li>
    <li><img src="logo4.svg" alt="Company 4" /></li>
    <li><img src="logo5.svg" alt="Company 5" /></li>
  </ul>
</section>
```
