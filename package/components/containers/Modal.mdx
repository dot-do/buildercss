# Modal

A centered overlay dialog for focused interactions.

## Semantic Structure

```html
<dialog open aria-modal="true" aria-labelledby="modal-title">
  <article>
    <header>
      <h2 id="modal-title">Modal Title</h2>
      <button aria-label="Close" data-action="close">Ã—</button>
    </header>
    <section>
      <!-- Content -->
    </section>
    <footer>
      <button data-action="cancel">Cancel</button>
      <button data-action="confirm">Confirm</button>
    </footer>
  </article>
</dialog>
```

## CSS

```css
dialog {
  position: fixed;
  inset: 0;
  display: flex;
  align-items: center;
  justify-content: center;
  background: rgba(0, 0, 0, 0.5);
  backdrop-filter: blur(4px);
  z-index: 100;
}

dialog::backdrop {
  background: rgba(0, 0, 0, 0.5);
}

dialog > article {
  background: var(--background);
  border: 1px solid var(--border);
  border-radius: var(--radius-lg);
  max-width: 500px;
  width: 90vw;
  max-height: 90vh;
  overflow: auto;
}

dialog > article > header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: var(--spacing-4);
  border-bottom: 1px solid var(--border);
}

dialog > article > section {
  padding: var(--spacing-6);
}

dialog > article > footer {
  display: flex;
  justify-content: flex-end;
  gap: var(--spacing-2);
  padding: var(--spacing-4);
  border-top: 1px solid var(--border);
}
```

## Sizes

```css
dialog[data-size="sm"] > article { max-width: 400px; }

```

## Mobile (React Native)

```tsx
<Modal
  visible={visible}
  transparent
  animationType="fade"
  onRequestClose={onClose}
>
  <View style={styles.overlay}>
    <View style={styles.content}>
      <ModalHeader title={title} onClose={onClose} />
      <ScrollView>{children}</ScrollView>
      <ModalFooter onCancel={onCancel} onConfirm={onConfirm} />
    </View>
  </View>
</Modal>
```

## Props

```typescript
interface ModalProps {
  open: boolean
  onClose: () => void
  title?: string
  size?: 'sm' | 'md' | 'lg' | 'xl' | 'full'
  showClose?: boolean
  children: ReactNode
  footer?: ReactNode
}
```
