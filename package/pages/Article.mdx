# Article

An individual blog post or article page.

## Composition

```
StackedLayout
├── ArticleHeader (title, meta, author)
├── ArticleContent (prose)
├── ArticleFooter (tags, share, related)
└── Comments (optional)
```

## Semantic Structure

```html
<body>
  <header><!-- Nav --></header>
  <main>
    <article>
      <header>
        <time datetime="2025-01-15">January 15, 2025</time>
        <h1>Article Title</h1>
        <p>Article description/excerpt</p>
        <address>
          <a href="/author/john">John Doe</a>
        </address>
      </header>
      <section>
        <!-- Prose content -->
      </section>
      <footer>
        <aside aria-label="Tags">
          <a href="/tag/react">React</a>
          <a href="/tag/nextjs">Next.js</a>
        </aside>
        <nav aria-label="Share">
          <a href="#">Twitter</a>
          <a href="#">LinkedIn</a>
        </nav>
      </footer>
    </article>
    <aside aria-label="Related posts">
      <h2>Related Posts</h2>
      <RelatedPosts />
    </aside>
  </main>
  <footer><!-- Footer --></footer>
</body>
```
