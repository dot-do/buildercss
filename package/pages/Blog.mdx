# Blog

A blog listing page showing all posts with optional filtering.

## Composition

```
StackedLayout
├── Header (title, description)
├── PostList (CardView or ListView)
└── Pagination
```

## As Page vs Section

**As Page**: Full blog index at `/blog`
**As Section**: Recent posts widget on homepage

## Semantic Structure

```html
<body>
  <header><!-- Nav --></header>
  <main>
    <header>
      <h1>Blog</h1>
      <p>Latest news and updates</p>
    </header>
    <section aria-label="Posts">
      <article>
        <time datetime="2025-01-15">Jan 15</time>
        <h2><a href="/blog/post">Post Title</a></h2>
        <p>Excerpt...</p>
        <aside><span>tag1</span></aside>
      </article>
      <!-- More posts -->
    </section>
    <nav aria-label="Pagination">
      <a href="/blog?page=1">1</a>
      <a href="/blog?page=2" aria-current="page">2</a>
    </nav>
  </main>
  <footer><!-- Footer --></footer>
</body>
```
