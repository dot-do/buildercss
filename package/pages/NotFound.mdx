# NotFound

A 404 page for when content is not found.

## Semantic Structure

```html
<body>
  <header><!-- Nav --></header>
  <main>
    <section>
      <h1>404</h1>
      <p>Page not found</p>
      <p>The page you're looking for doesn't exist.</p>
      <nav>
        <a href="/">Go home</a>
        <a href="/search">Search</a>
      </nav>
    </section>
  </main>
  <footer><!-- Footer --></footer>
</body>
```
