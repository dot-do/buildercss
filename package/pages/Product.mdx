# Product

A product page highlighting features, benefits, and use cases.

## Composition

```
StackedLayout
├── Hero (product intro)
├── Features (detailed features)
├── HowItWorks (steps)
├── UseCases (examples)
├── Testimonials (social proof)
└── CTA (get started)
```

## Semantic Structure

```html
<body>

  <main>

        <img src="screenshot.png" alt="Product screenshot" />
      </figure>
    </section>
    <section aria-label="Features">
      <h2>Features</h2>

  </main>
  <footer><!-- Footer --></footer>
</body>
```
