# Changelog

A changelog page showing version history.

## Semantic Structure

```html
<body>
  <header><!-- Nav --></header>
  <main>
    <article>
      <header>
        <h1>Changelog</h1>
        <p>All notable changes to this project</p>
      </header>
      <section>
        <article id="v2.0.0">
          <header>
            <h2>v2.0.0</h2>
            <time datetime="2025-01-15">January 15, 2025</time>
          </header>
          <section aria-label="Added">
            <h3>Added</h3>
            <ul><li>New feature...</li></ul>
          </section>
          <section aria-label="Changed">
            <h3>Changed</h3>
            <ul><li>Updated...</li></ul>
          </section>
          <section aria-label="Fixed">
            <h3>Fixed</h3>
            <ul><li>Bug fix...</li></ul>
          </section>
        </article>
        <!-- More versions -->
      </section>
    </article>
  </main>
  <footer><!-- Footer --></footer>
</body>
```
