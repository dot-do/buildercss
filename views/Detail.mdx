# Detail

A detail view for displaying a single item with all its properties.

## Semantic Structure

```html
<article aria-label="Item detail">
  <header>
    <nav aria-label="Breadcrumb">
      <a href="/items">Items</a> / <span>Item Name</span>
    </nav>
    <h1>Item Name</h1>
    <menu>
      <button>Edit</button>
      <button>Delete</button>
    </menu>
  </header>

  <section aria-label="Properties">
    <dl>
      <div>
        <dt>Status</dt>
        <dd><span data-status="active">Active</span></dd>
      </div>
      <div>
        <dt>Created</dt>
        <dd><time>January 15, 2025</time></dd>
      </div>
      <div>
        <dt>Owner</dt>
        <dd><a href="/users/john">John Doe</a></dd>
      </div>
    </dl>
  </section>

  <section aria-label="Description">
    <h2>Description</h2>
    <p>Item description content...</p>
  </section>

  <section aria-label="Related">
    <h2>Related Items</h2>
    <ul><!-- Related items --></ul>
  </section>
</article>
```
