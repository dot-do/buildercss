# Calendar

A calendar view for displaying and managing events.

## Semantic Structure

```html
<section aria-label="Calendar">
  <header>
    <h1>Calendar</h1>
    <nav aria-label="Calendar navigation">
      <button aria-label="Previous month">←</button>
      <h2>January 2025</h2>
      <button aria-label="Next month">→</button>
    </nav>
    <menu>
      <button aria-pressed="true">Month</button>
      <button>Week</button>
      <button>Day</button>
    </menu>
  </header>

  <table role="grid">
    <thead>
      <tr>
        <th>Sun</th>
        <th>Mon</th>
        <th>Tue</th>
        <th>Wed</th>
        <th>Thu</th>
        <th>Fri</th>
        <th>Sat</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td><time datetime="2025-01-01">1</time></td>
        <td>
          <time datetime="2025-01-02">2</time>
          <article aria-label="Event">
            <span>Meeting</span>
          </article>
        </td>
        <!-- More days -->
      </tr>
    </tbody>
  </table>
</section>
```
